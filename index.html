Core Capabilities

simulate the physical world

Real-Time Simulations

Run complex physics simulations at interactive frame rates. Watch particles, forces, and fields evolve in real time with sub-millisecond response — no pre-baking required.

📈

Graph Plotting

Render publication-ready plots from live simulation data. Supports Cartesian, polar, parametric, phase portrait, and custom coordinate systems with SVG/PNG export.

Scientific Calculator

Evaluate symbolic and numerical expressions, derivatives, integrals, and matrix operations — all tightly integrated with the live simulation environment.

Vector Visualization

Display vector fields, force diagrams, and gradient flows as interactive arrows, stream lines, or heat maps overlaid directly on your simulation canvas.

Physics Experiments

Parameter Control
